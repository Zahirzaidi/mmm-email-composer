# MMM Email Composer

Web-based email composer for Majlis Media Malaysia.

## Features

- Select recipients from Supabase categories, groups, and members.
- Compose email content in Markdown.
- Preview email with the official MMM template.
- Send email through Resend.
- Track sent, opened, unread, bounced, and failed emails.
- View history, drafts, and scheduled email records.
- Protect admin pages with a password login.

## Deploy

This project is ready for Render as a Node web service.

Use:

```bash
npm install
npm start
```

Required production environment variables are listed in `.env.example`.

See `DEPLOYMENT.md` for the full checklist.
